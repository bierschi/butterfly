# Ransomware that makes you cry


## Installation
<pre><code>
git clone https://github.com/bierschi/ransomware.git
cd ransomware
mkdir build
cmake ..
make
</code></pre>


## Usage and Examples

Print the available arguments for Ransomware
<pre><code>
ransomware --help
</code></pre>


## Logs

logs can be found in `/var/log/ransomware`
