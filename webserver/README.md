# bflyServerApp
Provides the Route to decrypt the Clients CPrivateRSA.bin file

## Installation

<pre><code>
git clone https://github.com/bierschi/butterfly.git
cd bflyServerApp
sudo python3 setup.py install
</code></pre>

## Usage and Examples
Print the available arguments
<pre><code>
bflyServerApp --help
</code></pre>
